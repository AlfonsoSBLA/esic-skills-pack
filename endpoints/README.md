## Cómo se usan

Cada endpoint es un archivo markdown que contiene:
- La URL o credencial necesaria
- Instrucciones de uso paso a paso
- Limitaciones (rate limits, datos de prueba, etc.)
- Plan B si el endpoint no responde

Estos endpoints están bloqueados al alumnado mientras no estés en clase — se publican el día correspondiente.
